스펙 §6.1 기준(라이트 UI · Pretendard · 바이올렛 #6D4AFF · Notion계 미니멀). 첨부한 레퍼런스(Snapdeck류)의 좋은 패턴 — Chat/속성 탭 전환, 생성 중 슬라이드 플레이스홀더, 그리드 오버뷰 — 은 가져오되, 가독성 문제를 교정했습니다: 본문 최소 13px, 라벨에 텍스트+색 병용(아이콘 단독 금지), 저대비 회색 텍스트 제거, 상태는 배지로 명시. 화면당 2안 — A안 = 스펙 정석 레이아웃, B안 = 패널 배치·AI 인터랙션(채팅 vs 인라인)·톤 변형. 예시 콘텐츠는 "소상공인 경영바우처 지원 제안서" 5장 덱.
