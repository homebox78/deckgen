# 이모지(이모티콘) 이미지 드롭 폴더

무료 이모티콘 이미지를 **시리즈별 하위 폴더**로 여기에 넣어주세요. 그러면 서버에 업로드하고
DeckGen 에디터의 "이모지" 탭에서 시리즈별로 골라 슬라이드에 삽입할 수 있게 연결합니다.

## 규칙
```
assets/emoji/
├── <시리즈명1>/        예) toss-face, sticker-pack-a
│   ├── smile.png
│   ├── cry.png
│   └── ...
├── <시리즈명2>/
│   └── ...
```
- 폴더명 = 시리즈명(영문/숫자/하이픈 권장, 탭 라벨로 표시)
- 파일 = PNG(투명 배경) 또는 GIF(움직이는 이모티콘). 파일명 = 이모지 이름(툴팁으로 표시)
- 권장 크기: 128~512px 정사각. 너무 크면 업로드 시 자동 축소

## 업로드
파일을 넣은 뒤 "이모지 등록해줘" 라고 하면:
1. 시리즈별로 서버(`hom2box.com/deckGen/emoji/<시리즈>/`)에 업로드
2. 매니페스트(`/emoji/manifest.json`) 갱신
3. 에디터 "이모지" 탭에 **시리즈 선택 + 이미지 그리드**로 노출 → 클릭 시 이미지로 삽입

라이선스가 자유로운(무료·상업적 사용 가능) 이모티콘만 넣어주세요.
